# Project Summary
UCars Motors is a comprehensive web application tailored for the automotive industry, focusing on car trading, rentals, and parts sales. It acts as a digital marketplace where users can browse vehicles, estimate financing options, and inquire about parts and salvage vehicles. The platform aims to enhance user engagement through a modern interface, responsive design, and effective SEO strategies. Recent updates have introduced an admin panel for managing vehicle listings, parts, and company contact information.

# Project Module Description
- **Homepage**: Introduces users to the platform and its services.
- **Vehicles**: Displays a catalog of available vehicles with advanced filtering options.
- **Calculator**: Offers a financing calculator for estimating monthly payments.
- **Parts**: Lists automotive parts for sale with search capabilities.
- **Salvage**: Showcases damaged vehicles available for parts or restoration.
- **Contact**: Provides a way for users to reach out for inquiries and support, displaying updated company contact information.
- **Admin Panel**: A password-protected interface for managing vehicles, parts, and company details, allowing users to add, edit, and delete entries.

# File Description Inventory
- **.gitignore**: Specifies which files and directories to ignore in version control.
- **README.md**: Overview and installation instructions.
- **package.json**: Lists dependencies and scripts for project execution.
- **src/App.tsx**: Main application file that sets up routing and context providers.
- **src/pages/**: Contains individual page components like Index, Vehicles, Parts, Admin, and Contact.
- **src/components/**: Reusable UI components such as Navigation and Footer.
- **src/pages/Admin.tsx**: Admin panel for managing vehicles, parts, and company details.
- **src/pages/Contact.tsx**: Contact page displaying company information and a form for inquiries.
- **src/components/Footer.tsx**: Footer component displaying company contact details and operational hours.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript providing type safety.
- **Vite**: Build tool for rapid development and production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: Library for managing routing in React applications.
- **TanStack Query**: For efficient data fetching and caching.

# Usage
1. **Install Dependencies**: Run `pnpm install` to install project dependencies.
2. **Build the Project**: Execute `pnpm run build` to create a production build.
3. **Run the Application**: Use `pnpm run dev` to start the development server.
